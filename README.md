# 毛选思想指导 (ChairmanMao-guide-life)

以毛泽东选集核心思想指导判断和行动的 VS Code 智慧对话扩展。

## 功能特性

- **六阶段结构化分析**：全面了解 → 主要矛盾 → 条件评估 → 战略建议 → 战术行动 → 反思迭代
- **毛选思想框架**：实事求是、矛盾论、实践论、群众路线、游击战术、统一战线、自我革新
- **多风格融合**：可选偏重毛选原教旨主义、叶子农/丁元英方法论，或平衡融合
- **API 可配置**：默认优先支持 DeepSeek，兼容 OpenAI API 格式
- **对话历史管理**：自动存档、随时回溯、删除管理
- **分析报告导出**：一键生成 Markdown 格式的结构化分析报告
- **联网搜索**：可选集成 SerpAPI / Bing Search API

## 十项核心思想

1. **认清形势** — 分清敌我，方向才对
2. **没有调查就没有发言权** — 重事实，不空谈
3. **抓主要矛盾** — 抓住关键，全局盘活
4. **战略与战术结合** — 战略敢赢，战术做实
5. **群众路线** — 把目标变成大家的利益
6. **游击战术** — 避实击虚，不硬碰硬
7. **在实践中学习** — 边干边学，知行合一
8. **实事求是** — 一切从实际出发
9. **统一战线** — 团结多数，孤立少数
10. **自我革新** — 常自省，常更新

## 安装

1. 下载 `.vsix` 文件
2. 在 VS Code 中打开命令面板 (`Ctrl+Shift+P`)
3. 选择 **Extensions: Install from VSIX...**
4. 选择下载的 `.vsix` 文件

## 使用方法

1. 点击侧边栏 **毛选思想指导** 图标
2. 点击 **设置齿轮** 配置 API Key（默认支持 DeepSeek）
3. 点击 **新建会话** 开始对话
4. 向毛选思想分身描述你面临的问题
5. 扩展会引导你通过六阶段逐步深入分析
6. 对话完成后可**导出报告**

## API 配置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| API 服务地址 | OpenAI 兼容 API 端点 | `https://api.deepseek.com` |
| API Key | 你的 API 密钥 | - |
| 模型名称 | 使用的模型 | `deepseek-chat` |
| Temperature | 输出随机性 (0-2) | `0.7` |
| 最大 Token 数 | 单次回复最大长度 | `4096` |

推荐使用 [DeepSeek](https://platform.deepseek.com/) 获取 API Key。

## 对话风格

- **平衡融合（推荐）**：自然地融合毛选思想与叶子农/丁元英方法论
- **偏重毛选原教旨主义**：更多直接引用毛泽东选集原文
- **偏重叶子农/丁元英方法论**：更多使用"见路不走"、"文化属性"概念框架

## 开发

```bash
npm install        # 安装依赖
npm run compile    # 开发构建
npm run watch      # 监听模式
npm run package    # 打包 .vsix
```
